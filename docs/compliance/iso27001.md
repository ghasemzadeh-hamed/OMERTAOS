# ISO 27001 Controls Overview

- Information security policy maintained and version controlled.
- Asset management registers linked to governance dashboard metrics.
- Access control implemented via RBAC policies enforced in kernel AIRouter.
- Cryptography relies on TLS termination with secret rotation policies.
- Operations security includes continuous monitoring via observability stack.
- Supplier relationships vetted before integration layering is configured.
- Incident response playbooks wired into CLI backup and restore flows.
- Business continuity validated by deploy/scripts/backup.sh automation.
- Compliance evidence exported through governance dashboard APIs.
